# Lab 02 — Code Skeleton

The Lab 02 statement is in `docs/academics/lab-02-riscv-isa-primer.md`.
This directory contains the small code artefacts the lab refers to.

The lab is mostly hand-encoding exercises: take a snippet of RV32I
assembly and produce the 32-bit binary encoding by reading the
RISC-V Unprivileged ISA manual.

Once you have your encodings, drop them into `exercises.c` (below)
and run `make test`. The autograder checks each encoding against the
reference.
